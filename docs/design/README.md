## Componentes

### Botões
- **Primário**: Cor Primary 500, hover Primary 600
- **Secundário**: Cor Gray 500, hover Gray 600
- **Tamanho**: Padding 0.75rem 1.5rem, Rounded 0.5rem

### Cards
- **Background**: Branco
- **Border**: Gray 200
- **Shadow**: `shadow-lg`
- **Padding**: 2rem

### Inputs
- **Border**: Gray 300
- **Focus Ring**: Primary 500
- **Padding**: 0.5rem 1rem
- **Rounded**: 0.5rem

## Responsividade

### Breakpoints
- **Mobile**: < 768px
- **Tablet**: 768px - 1024px
- **Desktop**: > 1024px

### Grid
- **Mobile**: 1 coluna
- **Tablet**: 2 colunas
- **Desktop**: 3-4 colunas

## Animações

### Transições
- **Hover**: `transition-colors duration-200`
- **Focus**: `transition-all duration-200`

### Micro-interações
- **Button Hover**: Escurecer cor em 10%
- **Card Hover**: Aumentar sombra
- **Input Focus**: Mostrar ring colorido
